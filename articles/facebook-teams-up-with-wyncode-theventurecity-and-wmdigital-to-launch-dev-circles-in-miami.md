# Facebook teams up with Wyncode, TheVentureCity and WM.Digital to launch Dev Circles in Miami

![Wyn1](../images/6a00d83451b26169e201bb09bdc9fd970d-800wi.jpg "Wyn1")

**By Juha Mikkola**

Last week, Facebook teamed up with Wyncode Academy, TheVentureCity and WM.Digital to launch the first Developer Circles in North America.

  ![Wyn2](https://web.archive.org/web/20170903060456im_/http://miamiherald.typepad.com/.a/6a00d83451b26169e201bb09bdca2d970d-800wi "Wyn2")

Facebook’s Dev Circles empower a diverse range of technical talent to improve their skills and access the information they need.

The sold-out event at Wynbase brought together over 100 developers of varying levels to learn about React. React is a JavaScript library created by Jordan Walke, a software engineer at Facebook. Wyncode teaches React as part of its 10 week immersive course and in its new part-time Front End Web Development course.

![Wyn3](https://web.archive.org/web/20170903060456im_/http://miamiherald.typepad.com/.a/6a00d83451b26169e201b8d2a4f884970c-800wi "Wyn3")  
 

The speakers included:

\* [Chris Stegner](https://www.linkedin.com/in/chris-stegner-30175825/) of MadDev, who was [selected](http://www.miamichamber.com/events/awards-programs/technology-leaders-year-awards) Technology Entrepreneur of the Year by the Greater Miami Chamber of Commerce and has hired a Wyncoder Developer, spoke about why his firm uses Reac

\* [Adrian Martin](https://www.linkedin.com/in/mradrianmartin/) of CareCloud, who have hired 13 Wyncode Developers, presented React JS basics

\* Wyncode’s own [Ed Toro](https://www.linkedin.com/in/eddroid/) live coded how to add React into an existing software stack using Rails as an example
