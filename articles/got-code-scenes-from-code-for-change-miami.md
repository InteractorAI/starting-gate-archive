# Scenes from Hack for Change: Miami

Got Code? South Florida techies got their code on over the weekend at the [Hack for Change: Miami](http://www.hackforchangemiami.org/) event at [The LAB Miami](http://www.thelabmiami.com/). At the same time, citizens, software developers and entrepreneurs across the nation joined forces to celebrate a National Day for Civic Hacking to create solutions using publicly-released data sets and code. Teams worked tirelessly over two days to solve challenges relevant to their communities. Here are a few scenes the event event Saturday,  thanks to [Theo Karantsalis](http://www.linkedin.com/profile/view?id=188507492&locale=en_US&trk=tyah), who provided the photos and descriptions.
