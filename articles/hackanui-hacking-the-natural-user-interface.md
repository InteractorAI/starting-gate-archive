# HackaNUI - Hacking the Natural User Interface

[![Bgimg2](../images/6a00d83451b26169e201910239fc01970c-800wi.jpg "Bgimg2")](http://miamiherald.typepad.com/.a/6a00d83451b26169e201910239fc01970c-pi)

Forget about web and mobile, those are so yesterday. Natural User Interface (NUI) applications use gestures and motion to interact with applications. The technology is so exciting that a hackathon has been created around it.

[![HackaNUILogo](https://web.archive.org/web/20140728161204im_/http://miamiherald.typepad.com/.a/6a00d83451b26169e201901c43fea1970b-250wi "HackaNUILogo")](http://miamiherald.typepad.com/.a/6a00d83451b26169e201901c43fea1970b-pi)HackaNUI is a first of its kind hackathon dedicated to creating applications centered around Natural User Interface (NUI) technologies. Microsoft and PrimeSense (the makers of the Kinect technology) are sponsoring the event as well as many other prominent organizations.

So what is HackaNUI exactly? Mix in some awesome technology, some really talented people, throw in a day of hacking, food, fun and finally prizes and there it is!

The first HackaNUI is set for Saturday June 1, 2013 at Caffeine Spaces in Boca Raton and is free. So far, about 150 people have signed up, said Marc Asselin, founder of mLabs.
